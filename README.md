## 🖥️ What this app does

ai-video-captions adds animated subtitles to your videos. It can turn spoken words into captions, then place them on screen in styles that fit short-form video.

It works with:
- TikTok clips
- Instagram Reels
- YouTube Shorts
- Talking-head videos
- Tutorials
- Product clips

It uses Whisper for speech-to-text and FFmpeg for video processing. That means it can read the audio in your video, create subtitles, and render them back into the file.

## ✨ Main features

- 6 caption styles, including Hormozi, MrBeast, and Karaoke

## 🔍 Troubleshooting

### The app cannot read my video

- Make sure the file is not damaged
- Try a simple `.mp4` file first
- Move the file to a folder with a short path, such as `C:\Videos`

### The captions are wrong

- Check that you chose the correct language
- Make sure the audio is clear
- Reduce background noise if possible
- Try a different video with cleaner speech

### The export is slow

- Longer videos take more time
- Close other apps while processing
- Use a shorter test clip first
- Check that your PC has enough free memory

### The output has no audio

- Make sure the source video had audio
- Check the export settings
- Try a different input file
